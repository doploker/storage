# Как установить

```bash
git clone https://github.com/doploker/storage.git
```

```bash
python -m venv .venv
```

```bash
.venv\Scripts\activate
```

```bash
pip install -r requirements.txt
```

```bash
py manage.py makemigrations
```

```bash
py manage.py migrate
```

```bash
py manage.py createsuperuser
```

```bash
py manage.py runserver
```
